# Calculator

This project is a simple browser-based on-screen calculator built with JavaScript.

## Technology used 

- HTML5
- CSS
- JavaScript
- Visual Studio Code
- Git 

## Features

- Basic math operations
- Keyboard support
- Clear / delete buttons
- Responsive layout
- Error handling 

## About the project

The goal of this project is to practice DOM manipulation, event handling, and 
writing reusable functions while building a real-world interactive application.

## What I learned

- DOM manipulation
- Event handling
- Functions and modular code
- State management
- Basic testing and debugging
